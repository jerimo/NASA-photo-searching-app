### 🚀 NASA API를 이용한 사진 검색 서비스입니다.

-----

- 흥미 유발을 위해 'Galaxy'를 검색했을 때의 결과가 초기 값으로 보여지며, 검색어를 입력할 수 있습니다.

- 데이터는 9개씩 보여지며 스크롤이 최하단으로 내려가면 다시 9개의 데이터를 불러옵니다.
